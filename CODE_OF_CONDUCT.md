# Code of Conduct

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/version/2/1/code_of_conduct/),
version 2.1, as its Code of Conduct.

In short: be respectful and constructive. Personal attacks, harassment, and discrimination
are not tolerated. Contributions are welcome from everyone, regardless of background or
experience level.

## Scope

This applies to all project spaces — the repository, issues, pull requests, and any other
official channel — and whenever an individual is representing the project in public.

## Enforcement

Report unacceptable behavior privately through
[GitHub Security Advisories](https://github.com/joshuamatosdev/do/security/advisories/new)
(the "Report a vulnerability" channel doubles as a private contact for conduct concerns).
Reports are reviewed promptly and fairly, and the privacy of the reporter is respected.

The full standards, enforcement responsibilities, and enforcement guidelines are those of
the Contributor Covenant v2.1, linked above.
