# Memory Index

One line per memory: `- [Title](file.md) — hook`. Memory bodies live in files next to this one, never here.

A memory file holds ONE fact with YAML frontmatter: `name` (kebab-slug), `description` (one-line summary for recall), and `metadata.type` — one of `user` | `feedback` | `project` | `reference`. The fact follows the frontmatter; link related memories with [[their-name]].
